# Sierpinski-triangle
This is 3d implementation of Sierpinski-triangle. You can move around this figure by using keys:

* **W** - look up
* **A**  - look left
* **S**  - look down
* **D**  - look right
* **l**  - zoom out
* **o**  - zoom in
